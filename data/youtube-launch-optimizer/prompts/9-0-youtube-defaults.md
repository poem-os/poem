# YouTube Upload Defaults

Reference document for static YouTube upload fields. These are brand standards that rarely change.

## Details Tab

| Field              | Default Value      | Notes                                        |
| ------------------ | ------------------ | -------------------------------------------- |
| Audience           | Not made for kids  | Always select this                           |
| Age restriction    | No, don't restrict | Unless explicit content                      |
| Paid promotion     | No                 | Unless affiliate/sponsor disclosed           |
| Automatic chapters | Yes                | Override with manual chapters from Section 2 |

## Video Elements Tab

| Element    | Default                    | Notes                       |
| ---------- | -------------------------- | --------------------------- |
| Subtitles  | Auto-generated             | Add manual SRT if available |
| End screen | Add from template          | Use brand template          |
| Cards      | Add 2-3 related videos     | Use 9-3 prompt output       |
| Info cards | Add at relevant timestamps | Optional                    |

## Checks Tab

| Check          | Expected  | Action if Failed       |
| -------------- | --------- | ---------------------- |
| Copyright      | No issues | Review flagged content |
| Ad suitability | Suitable  | Review if limited      |

## Visibility Tab

| Field           | Default            | Notes                     |
| --------------- | ------------------ | ------------------------- |
| Save or publish | Public             | Or Schedule               |
| Schedule        | Thursdays 5pm AEST | Brand standard (flexible) |
| Premiere        | No                 | Unless special launch     |

## More Options

| Field                         | Default                                 | Notes                    |
| ----------------------------- | --------------------------------------- | ------------------------ |
| Category                      | Science & Technology                    | Brand standard           |
| License                       | Standard YouTube License                | Unless Creative Commons  |
| Allow embedding               | Yes                                     | Always                   |
| Publish to subscriptions feed | Yes                                     | Always                   |
| Comments                      | Show potentially inappropriate comments | Or "Hold all" for review |
| Show comment ratings          | Yes                                     | Default                  |
| Sort by                       | Top comments                            | Default                  |

## Brand-Specific Settings

### AppyDave Channel

- Category: Science & Technology
- Default playlists: Based on content (see 9-1 prompt)
- End screen template: Standard AppyDave template
- Comment moderation: Show all (trusted audience)

---

## When to Override Defaults

| Situation                          | Override                           |
| ---------------------------------- | ---------------------------------- |
| Tutorial with explicit code errors | May get "limited ads" - that's OK  |
| Collaboration video                | May need paid promotion disclosure |
| Time-sensitive content             | Schedule for immediate publish     |
| Educational series                 | Add to multiple playlists          |
