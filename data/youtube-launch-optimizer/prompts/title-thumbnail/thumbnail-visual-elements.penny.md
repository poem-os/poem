1. Main topic/theme
2. Key highlights
3. Emotional triggers

### 2. Triple-Set Output
**3 distinct visual concept sets**, each with:
- Description

**5 attributes per set** = rich concept definition.

### 3. Dual-Prompt Pattern
**Each set includes 2 prompts**:
- Primary visual element
- Secondary visual element

**Total**: 3 sets × 2 prompts = 6 visual prompts generated.

**Anti-complexity design principle** - aligns with 5-5's "avoid overly complex scenes."

### 5. Diversity Requirement
"Each set must be unique and offer a different approach"

**Forced variety** to provide choice.
